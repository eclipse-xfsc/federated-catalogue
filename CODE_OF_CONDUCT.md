# Code of Conduct

This project adopts
the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its Code
of Conduct. The canonical text — including standards, scope, enforcement guidelines, and reporting expectations — is
maintained at that URL.

> Community leaders may revise this document at any time to reflect the project's evolving needs.

## Eclipse Foundation framework

As an Eclipse Foundation project, conduct in this community is also governed by
the [Eclipse Code of Conduct](https://www.eclipse.org/org/documents/Community_Code_of_Conduct.php) and related
foundation policies.

## Reporting

Reports of unacceptable behavior should be sent to the Eclipse Foundation in accordance with
the [Eclipse Code of Conduct](https://www.eclipse.org/org/documents/Community_Code_of_Conduct.php#h.qe6n7vvn0svv)
reporting procedure. Reports are handled confidentially.
